Python for Astronomy Tutorial

Learn how to fetch and visualize real star data with Python

Step 1: Set Up Your Python Environment

Install Python from python.org. Then open your terminal, VS Code, or Jupyter Notebook and run:

pip install astropy astroquery matplotlib

Step 2: Import Libraries

These libraries will help fetch data and create plots:

from astroquery.simbad import Simbad
import matplotlib.pyplot as plt

Step 3: Query Star Data

Query stars in Orion within a 2-degree radius:

result = Simbad.query_region("Orion", radius="2 deg")
print(result[:5])

Step 4: Extract Coordinates

ra = result['RA']
dec = result['DEC']

Step 5: Plot the Stars

plt.figure(figsize=(8,6))
plt.scatter(ra, dec, color='blue', s=20)
plt.xlabel('Right Ascension (RA)')
plt.ylabel('Declination (DEC)')
plt.title('Stars in Orion')
plt.gca().invert_xaxis()
plt.show()

Step 6: Optional Challenge - Size by Brightness

brightness = result['FLUX_V']
plt.scatter(ra, dec, s=100/brightness, color='gold', alpha=0.7)
plt.xlabel('RA')
plt.ylabel('DEC')
plt.title('Stars in Orion (Sized by Brightness)')
plt.gca().invert_xaxis()
plt.show()

What You Learned
